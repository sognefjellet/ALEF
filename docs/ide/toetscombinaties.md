# Toetscombinaties

De meest gebruikte standaard toetencombinaties in ALEF. Deze zijn naar wens aan te passen via Settings > Keymap.

Windows | MacOS | Betekenis
------------ | ------------ | ------------- 
ALT + 1 | &#8997; + 1 | Roep de Projectbrowser op - hiërarchische structuur, links van het scherm, waarin alle elementen te vinden zijn.
ALT + 2 | &#8997; + 2 | Roep de Inspector op - venster waar, afhankelijk van de context, extra informatie te vinden is.
ALT + 3 | &#8997; + 3 | Roep Git - Version Control op - roep het venster op waarin te zien is wat gewijzigd is en daar (eventueel) acties op te nemen.
Ctrl + Up/Down | &#8997; + Up/Down | Maak selectie groter/kleiner (lijkt op Shift+ Up/Down maar selecteert anders (fijner)).
Ctrl + C / +V / +X | &#8984; + C / +V / +X | Standaard copy/paste/cut-functies. 
Ctrl + Z | &#8984; + Z | Undo. 
Ctrl + / | &#8984; + / | Comment/uncomment-toggle voor de selectie commentaar/geen commentaar.
Ctrl + Spatie | Cntrl + Spatie | Roep opties op om code/regel/definitie af te maken door element te selecteren (tik een paar letters om de selectielijst te beperken).
Alt + Enter | &#8997; + Enter | Roep Intentions op - soort contextgevoelig menu (lampje aan het begin van de regel).
Enter | Enter | Contextafhankelijke functie.
Shift + Up/Down | Shift + Up/Down | Maak selectie groter/kleiner (lijkt op Control + Up/Down maar selecteert anders (grover)).
Ctrl + D | &#8984; + D | Copy-functie voor volledige element waarop de cursor staat.
Ctrl + Shift + Z | &#8984; + Shift + Z | Re-do.
Ctrl + Shift + Up/Down | &#8984; + Shift + Up/Down | Verschuiven van volledige element waarop de cursor staat.

Voor een overzicht van alle [standaard toetscombinaties van MPS](https://www.jetbrains.com/help/mps/mastering-keyboard-shortcuts.html).
