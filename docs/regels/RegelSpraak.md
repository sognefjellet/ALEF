# RegelSpraak

Voor het opstellen van regels is de taal RegelSpraak ontwikkeld.

RegelSpraakregels kennen een vaste opbouw:

* **Naam**  
Iedere regel heeft een naam.
* **Regelversie**  
Regels bevatten één of meer regelversies met:
   * **Geldigheidsperiode**  
    Regelversies hebben een periode waarin ze geldig zijn.
   * **Regelinhoud**
       * [Resultaatdeel (actie)](../regels/Acties.md) (altijd aanwezig)
       * [Voorwaardendeel](./Voorwaardendeel.md) (optioneel)
       * [Variabelendeel](Variabelendeel.md) (optioneel)

![Een RegelSpraak regel met een actiedeel en een conditiedeel](../img/ALEF200_regel_actie_conditiedeel.png)

Naast regels kun je in RegelSpraak [beslistabellen](Beslistabel.md) gebruiken. Een beslistabel is een presentatievorm om gelijkstellingen op te nemen in het regelmodel.
